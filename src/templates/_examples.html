Hands-on Examples

To experiment with Blockonomicon, some example fields and templates can be installed for reference. You can then view the example content by visiting /blockonomicon-example. To uninstall this example content, you can find an uninstall button below, if the example content is detected.

{% if craft.app.getSections().getSectionByHandle('blockonomiconExample') == null %} What will this do?
  • Creates three blocks in Blockonomicon's block folder: exampleBanner, exampleTaggedContent, and exampleExpander. By default this folder is craft/storage/bloconomicon/blocks.
  • Creates a template in Craft's template folder called _blockonomicon.html.
  • Creates a tag group called "Blockonomicon Example Tag Group".
  • Creates an asset volume called "Blockonomicon Example Asset Volume", using a local store within a blockonomicon-assets folder in the web root.
  • Creates a matrix with the name "Blockonomicon Matrix".
  • Attaches the example blocks to the example matrix, using the example tag group and asset volume for applicable fields.
  • Creates a single section named "Blockonomicon Example", with the route /blockonomicon-example, using the example template.
  • Attaches the example matrix to the example section.
  • Adds a few example tags and small placeholder assets.
  • Fills out the single with some test data to be rendered.
